第16讲 综合实践：设计TicTacToe游戏（一）¶

Lecture 16. Project: Design a Tic Tac Toe Game (Part1)

Problem 问题描述¶

Observe or play a Tic Tac Toe game, answer the following questions: 观察或者亲自玩一盘TicTacToe游戏，回答下面的问题：

  1. Describe the how the game is being played 描述这个游戏是怎么玩的？

  2. How can we mathematically describe the whole game:从数学的角度来描述这个游戏

    1. game board 游戏的棋盘

    2. player’s action in each round 玩家在每一个回合中的动作

    3. check the end of the game 判断游戏结束的条件

  3. try to describe the game board by programming 通过编程来描述棋盘

def display_board(board):
    # print 3 rows * 3 columns
    """
      -  -  -
      -  -  -
      -  -  -
    """
    print("===  board status ===")
    for i, pos_status in enumerate(board):
        print("{}".format(pos_status), end=" ")
        if i % 3 == 2:
            print()
    print()
    return

def get_input(board, player):
    is_input_valid = False
    while not is_input_valid:
        try:
            pos = int(input("{}'s input:".format(player)))
        except:
            pass
        if pos >= 0 and pos < 9:
            if board[pos] == '-':
                is_input_valid = True
                return pos
        print("Not valid, input again.", end=" ")

def is_somebody_win(board):
    i = 0
    while i < 3:
        if ((board[3*i] == board[3*i+1] == board[3*i+2]) and board[3*i] != '-'):
            print("row {} lines up".format(i))
            return True
        i += 1
    i = 0
    while i < 3:
        if ((board[i] == board[i+3] == board[i+6]) and board[i] != '-'):
            print("col {} lines up".format(i))
            return True
        i += 1

    if ((board[0] == board[4] == board[8]) and board[0] != '-') or \
       ((board[2] == board[4] == board[6]) and board[2] != '-'):
        print("\ or / lines up")
        return True

    return False


def is_board_full(board):
    return '-' not in board

第2讲 起个名字吧¶

Objective 学习目标¶

  • Understand what a variable is 理解什么是变量

  • The key to learn a programming language is to learn the syntax/format/style of the languange 学习一门编程语言的关键在于学习这门语言的语法/格式/风格

  • Python is a case sensitive programming language Python中大小写不同表示不同的名称

  • Special meaning of “=” in programming 等于号在Programming中的另一种意义：赋值

  • Use variable to refer a string or a number 用变量表示字符串或数字

  • Write beautiful Python codes 书写漂亮的Python代码

    • space between operator 操作符两边留一个空格

    • assign multiple variables in one line 在一行代码中给多个变量赋值

  • Use print to print the value of a variable 用print打印变量值

Math¶

  • \(+\), \(-\), \(\times\), \(\div\)

Did you know?¶

  • There are many programming languages, and Python is just one of them. Python language can almost do anything programming work you want to. 有许多种编程语言，Python只是其中的一种。Python语言几乎能帮你做任何你想做的关于编程的工作。
